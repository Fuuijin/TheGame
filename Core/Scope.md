# Scope

## Tiers

### MVP — Ship This
The game that can be finished by two people and is worth playing.

| Feature | Notes |
|---|---|
| Legacy succession (heir / stranger) | Core differentiator — must ship |
| World persistence across runs | Economy + factions must simulate between sessions |
| 40×40 km map with set biomes | Layout fixed, not procedural |
| Seasonal cycle (4 seasons, real effects) | Drives ecology and economy |
| Basic faction system (form, war, treaty) | Villages grow/shrink based on conditions |
| Player character survival loop | Medieval survival: food, shelter, danger |
| UE 5.7 on PC | Target platform |

### V2 — Add If Healthy
After MVP ships and the project is alive.

| Feature | Notes |
|---|---|
| Children as fully-simulated NPCs | Heir who grows up in your run |
| Advanced ecology (species populations) | Wildlife migration, predator/prey loops |
| Light physics simulation | Volumetric lighting tied to seasons/time |
| Village merging / disappearing | Emergent settlements |
| Detailed economy (price simulation) | Supply/demand curves per region |

### Dream — Honor But Park
Ideas that define the soul of the game. Too big for now.

| Feature | Notes |
|---|---|
| Fully procedural world generation | Set biomes first |
| Multi-generational dynasty play | Needs V2 child system |
| Modding support | Post-1.0 |
| Console ports | After PC |
