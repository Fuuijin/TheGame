# Player Character

The avatar through which the player experiences each life in the world.

---

## Character Identity
- Each run, the player inhabits a specific person (heir, ally, or stranger — see [[Legacy System]])
- Characters have: name, origin, starting skills, starting relations
- Characters are mortal — death triggers succession

## Core Attributes (Draft)
- **Physical:** Health, stamina, strength, endurance
- **Skills:** Combat, crafting, trade, social, survival — improve through use
- **Reputation:** Distinct per faction and settlement (see [[Relations]])
- **Knowledge:** What this character knows (locations, people, secrets) — heirs inherit partially

## Starting Conditions
| Succession Type | Starting Assets | Starting Knowledge | Starting Relations |
|---|---|---|---|
| Named heir | Partial inheritance | Good (heir's childhood) | Inherited (modified) |
| Ally | Role/title only | Limited to shared history | Partial overlap |
| Random stranger | Nothing | None | None (world marks still exist) |
